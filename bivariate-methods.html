Bivariate Methods

Self-Assessment Questions12

  1. Why do we need a test of statistical significance?
  2. Given an example of a one-sided, and of a two-sided test of significance.
  3. Can we use continuous variables for cross-tabulations?
  4. Why is there no dependent and independent variable in correlation analysis?
  5. Consider the causality framework of Week 5. To what extent are (a) correlation and (b) cross-tabulations able to establish causality?

Please stop here and don’t go beyond this point until we have compared notes on your answers.

Applied Exercises in R (Core)

  1. Open the European Social Survey data, wave 9. Name it conveniently and attach it
  2. Univariate Analysis (revising weeks 1-2)
    1. Identify and explore briefly variables about respondents’ happiness, subjective health, age and household’s income (we are going to look at their relationships later)
    2. What do the values of the income variable mean?
    3. What is the mode of ‘age’?
    4. Try to write a function which automatically determines the mode of a variable (more advanced, don’t worry if this beats you. We will discuss this in the seminar.)
    5. Check the purpose of function round (?round) and, if relevant, apply it appropriately to your results from now onward
    6. Calculate means, variances and standard deviations when relevant
    7. Find a suitable plotting function for all four variables
  3. Crosstabulations
    1. We suspect there is a relationship between health and happiness. State the null and alternative hypothesis. Take care to correctly specify the dependent and independent variable.
    2. Cross-tabulate health and happiness, using the table function (search with ‘?’ if needed)
    3. Apply the ‘addmargins’ function to the previous formula
    4. Can you read the results? Can you interpret them?
    5. Display a bivariate plot with the same (search with ‘?’ if needed). What kind of plot do you get? Does this help reading and interpreting?
    6. Convert the table into percentages. You can use function ‘cprop’ or ‘lprop’ from package ‘questionr’ (you can install the new package through the menu “Tools” in RStudio). Interpret the result. Improve the table as much as you can.
    7. Conduct a Chi2 test of significance and interpret the result.
    8. Reorder the levels of Health using the ‘factor’ function with levels argument. Then display the cross-tabulation again.
    9. Is the relationship still statistically significant?
  4. Correlation
    1. Calculate the Pearson correlation between health and happiness (you need to work only with the cases that have positive values for both variables, search the help file for this)
    2. Does the result confirm your previous conclusion?
    3. Is the correlation coefficient statistically significant? What does this mean?
  5. Further Application
    1. Form the null und alternative hypotheses for the relationship between
      1. happiness and household income
      2. happiness and age
    2. Repeat the crosstabulation and correlation exercises for these two pairs of variables to test your hypotheses.

Applied Exercises in R (Going Further)

There is a lot to choose from this week, take your pick. There is no need to do all of these. Aim for two sections.

Section 1

  1. Use the ‘prop.table’ function to look at happiness against health in a comparison between genders.Compare genders as precisely as possible.
  2. Does health drive happiness more for men or women?
  3. Identify the two countries with lowest and highest happiness
  4. Compare happiness distributions between these two countries
  5. What benefit do distributions have compared to just looking at means?

Section 2

  1. For this section use Wave 7 (2014). Name it conveniently and attach it.
  2. Calculate the Body Mass Index variable with BMI=weight/height.
  3. Compare mean and standard deviation of BMI between the UK and Spain
  4. Apply ‘t.test’ function to test the difference between the two means (note this is now a two-sample t-test)
  5. Have the inhabitants of one country a significantly better BMI? Do you know why?

Section 3

  1. For this section use Wave 7 (2014). Name it conveniently and attach it.
  2. Plot the age at which respondents completed full-time education against the same for their fathers (note that these variables are only valid for the UK sample).
  3. There might be outlying values – can you see them?
  4. Plot the same, excluding outlying values
  5. Is there a significant link between fathers and children regarding this variable?
  6. Is there a significant link between mothers and children?
  7. Are these links, if any, stronger for sons or for daughters? Can you interpret?
