Exercises7

Conceptual Exercises

  1. The mean weight of a bag of apples is 1 kg. The weight of bags is normally distributed around this mean with a standard deviation of 50g.
    1. Billy is looking for the heaviest bag possible and finds one that is 1082 g. What is the probability of finding a heavier bag?
    2. What is the probability that Billy will find a bag lighter than 870g?
    3. How would the results of a. and b. change if the standard deviation was only 40g? Why?
  2. Which z-value gives you a right-tail probability of 2.5%?
  3. With a z-value of 1.13 what is the right-tail probability in %?
  4. The average mark on a fictitious module was 62.3 with a standard deviation of 8.5. What was the probability to fail the module (pass-mark=50)?

R Exercises - Core

  1. Open the European Social Survey data, but this time use Wave 7 (2014) which you need to download from the ESS website. Name it conveniently and attach it.
  2. Univariate Statistics
    1. Identify height and weight variables
    2. Use an appropriate function to summarise them in a few lines
    3. Plot the two variables separately as boxplots
    4. Plot the two variables separately as histograms using function ‘hist’
    5. Adjust the bars to 5 cm and 5 kg (or as close to these numbers as you can)
    6. Discuss the advantage of various break values
    7. What are weight’s and height’s modes?
    8. Plot the same using the ‘density’ of each variable (density is a function that smoothes a distribution)
    9. Calculate relevant central values for both variables. Do they match your intuition?
  3. Sampling
    1. Execute the following three times, then explain what function ‘sample’ is:
    sample(1:20,5)
    1. Execute the following three times, then explain what function ‘set.seed’ is:
    set.seed(1)
sample(1:20,5)
    1. Select a random sample of 100 from the ESS data frame, using “set.seed(1)”.
  4. Sampling Distributions
    1. Generate a vector “w” containing 10000 normally distributed numbers with mean 0 and standard deviation 1. Convert the vector into a data frame “dfw”.
    2. Using ggplot, plot a histogram of variable w (hint: use “after_stat(density)”), and overlay the histogram with a red density curve. Label the graph appropriately.
    3. Create an empty vector, called “n30”. Write a function which draws 9000 samples of size 30 from variable w and stores the mean of each sample in vector “n30”. Convert the vector “n30” into a data frame.
    4. Using ggplot, plot a histogram of variable n30 (hint: use “after_stat(density)”), and overlay the histogram with a red density curve. Label the graph appropriately.
    5. Plot the two graphs we just created in a grid with two columns and 1 row. (Hint: use package ‘gridExtra’)
    6. Interpret the juxtaposition of these graphs.

R Exercises - Going Further

    1. Complete Section 4 of the Core Exercises.
    2. Repeat steps a-d for the following distributions:
      • A uniform distribution with min = 0, max = 1.
      • A beta distribution, Beta(2,5).
    3. What does the beta distribution represent?
    4. Create a graph in which each distribution type has a new row, and population and sampling distributions are placed next to each other.
